# Keyboard Shortcuts

## Tools

| Shortcut | Action |
|---|---|
| `V` | Switch to Select tool |
| `E` | Switch to Erase tool |

## Editing

| Shortcut | Action |
|---|---|
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + Shift + Z` | Redo (alternative) |
| `Ctrl + C` | Copy selected wells |
| `Ctrl + V` | Paste wells (offset +5 mm X, −5 mm Y into a new group) |
| `Delete` / `Backspace` | Remove selected wells |

## Canvas navigation

| Shortcut | Action |
|---|---|
| `Alt + Drag` | Pan the canvas |
| `Middle mouse + Drag` | Pan the canvas |
| `Scroll wheel` | Zoom in / out |
