# Align to Each Other

Requires **at least two wells** selected. The **first selected well** acts as the anchor;
all others snap to its axis.

| Button | Action |
|---|---|
| ↔ Align H | Move all selected wells to the same **Y coordinate** as the anchor |
| ↕ Align V | Move all selected wells to the same **X coordinate** as the anchor |

The anchor is shown above the buttons as a labelled badge (e.g. `Anchor: A1 (first selected)`).

!!! note
    Align H aligns horizontally — meaning all wells end up on the same horizontal line (same Y).
    Align V aligns vertically — all wells end up on the same vertical line (same X).
