# Anywarelab

**Anywarelab** is a browser-based custom labware designer for the [Opentrons](https://opentrons.com) liquid handling platform.
Design well plates, reservoirs, tube racks, and tip racks visually, then export a ready-to-use
**Opentrons Labware Schema v2** JSON file — no coding required.

---

## What can I do with it?

- Place circular and rectangular wells on a configurable footprint
- Configure depth, volume, shape, and bottom type per well
- Align and distribute wells with precision tools
- Export Opentrons-compatible JSON and PNG previews
- Import and edit existing labware definitions
